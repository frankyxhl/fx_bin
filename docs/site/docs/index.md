### File Operations

| Command | Description | Documentation |
|----------|-------------|----------------|
| `fx files` | Count files in directories | [`fx files`](../commands/files.md) |
| `fx size` | Analyze file/directory sizes | [`fx size`](../commands/size.md) |

### File Finding

| Command | Description | Documentation |
|----------|-------------|----------------|
| `fx ff` | Find files by keyword | [`fx ff`](../commands/ff.md) |
